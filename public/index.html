Hindi Jokes

This app is built using Node.js, Express, and MongoDB. It uses the Hindi Jokes API to fetch random jokes in Hindi language. The app is deployed on Render.

Api endpoints
https://hindi-jokes-api.onrender.com/jokes

How your responce looks like

How to use

You can use this API in your projects. Just make a GET request to the API endpoint and you will get a random joke in Hindi language. You can also use the API in your browser by visiting the API endpoint in your browser.

How to contribute

You can contribute to this project by adding more jokes to the jokes.json file. Go to the GitHub repository Add more jokes to the jokes.json file and then make a pull request.

Disclaimer

this is a open source project and you can use it for free. Jokes in this API are taken from popular content being shared on WhatsApp and other social media platforms.
